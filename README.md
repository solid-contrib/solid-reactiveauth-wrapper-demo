## How to run

### On the command line

- Run 
```bash
npm start
```

- then navigate to [localhost:8080](http://localhost:8080/)

### In WebStorm

- Debug (not run) the _Start & Debug_ run/debug configuration.

### In Visual Studio Code

- Hit `F5`
- or click [this link](command:workbench.action.debug.start) in the markdown editor preview
- or run the _Debug in Chrome_ launch configuration in _Run and Debug_.
